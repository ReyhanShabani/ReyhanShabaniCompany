Here is the code for drawing with turtle and random module.
first we define a recursive function for drawing the trees iin a random way.
``` Python Code
def tree(l, s, t):
    if l > 0.9 * t:
        turtle.color(0.372, 0.29, 0.054)
    else:
        turtle.color(0, random.uniform(0.4, 0.7), 0)
    turtle.pensize(s)
    turtle.forward(l)
    if l > t:
        a = random.randint(15, 35)
        f = random.uniform(0.6, 0.8)

        turtle.left(a)
        tree(l*f, s-1, t)
        turtle.right(a)

        turtle.right(a)
        tree(l*f, s-1, t)
        turtle.left(a)
    turtle.backward(l)
```

Then we use this this code for drawing the river.
Here we have a list of diffrent types of Blues. Then we use a loop for drawing the curved lines.
``` Python code
lst_blues = [ (0.0, 0.1, 0.3), (0.0, 0.2, 0.5), (0.0, 0.3, 0.7), (0.0, 0.4, 0.9), (0.1, 0.5, 1.0), (0.2, 0.6, 1.0), (0.3, 0.7, 1.0), (0.4, 0.8, 1.0), (0.5, 0.9, 1.0), (0.6, 1.0, 1.0) ]
for i in range(-50, 300, 5):
    river = turtle.Turtle()
    river.penup()
    river.setpos(-800, i)
    river.setheading(0)
    river.pencolor(random.choice(lst_blues))
    river.pensize(30)
    river.pendown()

    for j in range(20):
        r = random.randint(25, 27)
        b = random.randint(25, 27)
        for i in range(r):
            river.forward(2)
            river.right(2)

        for i in range(b):
            river.forward(2)
            river.left(2)
```

And for drawing the mountains we use this code.
``` Python Code
t = turtle.Turtle()
t.penup()
t.goto(-800, 200)
t.pendown()
t.color(0.3, 0.24, 0.03)
t.pensize(2)
for i in range(4):
    t.begin_fill()
    t.setheading(60)
    for i in range(120):
        t.forward(5)
        t.right(1)
    t.end_fill()
```
